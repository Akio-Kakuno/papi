NAME

PAPI_set_debug - set the current debug level for PAPI

SYNOPSIS
#include <papi.h>

int PAPI_set_debug(int debuglevel);

DESCRIPTION
PAPI_set_debug sets the default debug level for the PAPI library. Here debuglevel is one of the following constants as defined in the papi.h header file. The current debug level is internally stored in the PAPI library and is used by the default internal PAPI error handler subroutine.
