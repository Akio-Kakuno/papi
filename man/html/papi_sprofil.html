NAME

PAPI_profil, PAPI_sprofil - generate PC histogram data where hardware counter overflow occurs

SYNOPSIS
#include <papi.h>

int PAPI_profil(unsigned short *buf, unsigned bufsiz, unsigned long offset, unsigned scale, int EventSet, int EventCode, int threshold, int flags);

int PAPI_sprofil(PAPI_sprofil_t *prof, int profcnt, int EventSet, int EventCode, int threshold, int flags);

DESCRIPTION
PAPI_profil() takes the following arguments:

*buf -- pointer to profile buffer array.

bufsiz -- number of entries in *buf.

offset -- starting value of lowest entry in *buf

scale --

EventSet -- The PAPI EventSet to profile when it is started.

EventCode -- Code of the Event in the EventSet to profile.

threshold -- threshold value for the Event triggers the handler.

flags -- bit pattern to control the behavior of PAPI_sprofil().

PAPI_sprofil() takes the following arguments:

*prof -- pointer to sprofil structure.

profcnt --

EventSet -- The PAPI EventSet to profile when it is started.

EventCode -- Code of the Event in the EventSet to profile.

threshold -- threshold value for the Event triggers the handler.

flags -- bit pattern to control the behavior of PAPI_sprofil().
